When to use

The 3 colum content grid should be used for simplistic content. Anything within the grid will be split evenly between all columns, this therefore lends itself well to lists or text based content.

The class can be applied to a list or wrapping div. With a list being used in the live example.

Do not use this snippet for overall page layouts, it will not allow you to control what appears in which column.
