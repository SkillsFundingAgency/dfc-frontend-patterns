When to use

This snippet should be used when further information is seen as being beneficial to the citizen when using search. As with all enhancements on the GDS patterns, this will need to be backed up by appropriate user research.

There are three additional pieces of information being displayed here, the alternative title, salary and category links. You can adapt and remove these where neccessary to suit the content you wish to provide.
