When to use

This snippet replaces the GDS default search from right hand column pattern which can be seen on multiple gov.uk departmental sites. The ultimate aim of the snippet is to increase user engagement, on explore careers search from job profile is out key route to further job profiles from the first profile a citizen hits.

Dependencies

The SCSS used for the right hand column search widget relies on some of the GDS toolkit defaults for includes and extends. It's therefore worth making sure you're using both the toolkit and govuk elements before utilisting this snippet.

Research

Background

75% of user's traffic search a keyword on Google and land at one of our profiles. There was an eminent need to be able to browse and search for other profiles within the job profile page, hence the search box was introduced to cater for that need. Usually the search box is placed on the top right corner but we've observed that then users were associating it with a wider GOV.UK search rather than looking at job profiles.

Research finding

The interactive element was tested on a number of scenarios with no usability issues present.
