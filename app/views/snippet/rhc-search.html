When to use

This snippet replaces the GDS default search from right hand column pattern which can be seen on multiple gov.uk departmental sites. The ultimate aim of the snippet is to increase user engagement, on explore careers search from job profile is out key route to further job profiles from the first profile a citizen hits.

Dependencies

The SCSS used for the right hand column search widget relies on some of the GDS toolkit defaults for includes and extends. It's therefore worth making sure you're using both the toolkit and govuk elements before utilisting this snippet.
