About this site

This site was set up to provide access to reusable markup snippets for all projects sitting under the National Careers Service. The ultimate aim is to enable collaborative working to better take place between all designers on all projects whilst maintaining design and markup consistency on citizen facing services.

Before you begin

All markup referenced on Explore careers snippets has a number of dependencies. You cannot simply grab the HTML, compile the SCSS and have everything working.

So before you begin you'll need to do the following

Install and reference the govuk template

Firstly you'll need to install and reference the govuk template. This includes some of the default styling for the top navigation section and footer of your product. Alongside this it comes with elements like the govuk cookie javascript which are referenced in some of the snippets.

Further information and an installation guide can be found on https://github.com/alphagov/govuk_template

Install and reference the alpha govuk front end toolkit

On Explore careers we use NPM to install any dependencies then reference these in our SCSS / HTML where required.

Further information and an installation guide can be found on https://github.com/alphagov/govuk_frontend_toolkit

Install and reference govuk elements

As with the govuk toolkit, we install elements using NPM then reference it in our SCSS. You'll also need to do this prior to using Explore Careers Snippets.

Further information and an installation guide can be found on https://github.com/alphagov/govuk_elements

Grab the Explore Careers base SCSS

There will be numerous references to this in the SCSS snippets found accross the site. In order for your SCSS snippets to compile this will need to be referenced before them.

You can download the ExploreCareers-Base SCSS file here.

Note: At the top of this file you'll find references to various GOVUK toolkit and elements SCSS files, you'll need to update the location these are being imported from.
