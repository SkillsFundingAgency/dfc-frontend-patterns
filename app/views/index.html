A library for design patterns to be used across the National Careers Service, to ensure consistency across the service and meet user needs where the GOV.UK design system does not.

This design library is created to store patterns which are used within the National Careers Service. For new product development, teams should look to make use of this design library along with the GOV.UK design system to meet user needs.

To understand how to utilise these patterns, you can check how to get started.
