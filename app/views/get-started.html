Import our styles from the CDN

The National Careers Service has a dedicated assets repository, which is deployed to a CDN. Here, we host all the assets required to run our digital services. To get started, you will need to reference our stylesheets and scripts.

The CSS file required is named all.min.css - this is a minified version which references GOV.UK styles and our own customised styles. The JavaScript file is all.min.js.

You can view the repository on GitHub, where you read about how to reference the files in the README file.
